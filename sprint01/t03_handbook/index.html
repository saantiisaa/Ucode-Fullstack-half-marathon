PROTOCOLS

HTTP Methods

The HTTP methods define actions to be performed on a resource, such as GET, POST, PUT, DELETE, and others

GET - retrieves information from the server
POST - sends information to the server
PUT - updates an existing resource
DELETE - deletes a resource
HEAD - retrieves the headers for a resource
OPTIONS - retrieves the supported methods for a resource
CONNECT - establishes a network connection to a resource
TRACE - retrieves a diagnostic trace of a resource

HTTP Status Codes

HTTP status codes are used to indicate the status of a client's request to a server

1xx - Informational
2xx - Successful
3xx - Redirection
4xx - Client Error
5xx - Server Error

DNS List by Types

DNS (Domain Name System) is used to translate domain names into IP addresses

A Record - maps a domain name to an IP address
AAAA Record - maps a domain name to an IPv6 address
CNAME Record - maps an alias name to a canonical name
MX Record - maps a domain name to a mail exchange server
NS Record - maps a domain name to a name server
PTR Record - maps an IP address to a domain name
SOA Record - specifies information about a zone or domain
TXT Record - adds additional information about a domain

Identifying Google's IP

One way to identify Google's IP address is by using the command prompt on a Windows computer. To do this, follow these steps:

  1. Open the command prompt
  2. Type "nslookup google.com" and press enter
  3. Look for the "Address" field in the output

The IP address associated with Google.com should be listed in the "Address" field.
